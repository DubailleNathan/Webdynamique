HTTP/1.1 est la version du protocole HTTP. Host: www.example.com spécifie l'hôte auquel vous faites la demande. Content-Type indique le type de contenu envoyé dans le corps de la requête. Il peut s'agir de application/x-www-form-urlencoded, application/json, multipart/form-data, etc., en fonction du type de données que vous envoyez. Content-Length indique la longueur des données envoyées. Le corps de la requête contient les données à envoyer, généralement sous la forme de paires clé-valeur pour les données de formulaire.
